# ColorPickerView
A color picker view for Android
Android颜色选择器

----

# 使用方法

## 1. 项目依赖

项目根目录下`build.gradle`中加入：
```
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```
添加依赖：
```
dependencies {
    implementation 'com.github.daemon369:ColorPickerView:v0.0.6'
}
```
